

# Mi Sitio Personal

Este proyecto es un sitio web personal construido con HTML y CSS. 
Utiliza una estructura semántica con `header`, `nav`, `main` y `footer`, además de secciones para Inicio, Sobre mí y Proyectos.

El objetivo es mostrar una base de sitio responsive y ordenada, con buena organización de archivos (pages/ y assets/), y contenido básico que representa un portfolio.

## Cómo verlo
Abrir `index.html` y, si querés, iniciar Live Server desde VS Code.
